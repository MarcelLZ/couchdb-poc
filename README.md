# CouchDB POC

Just a simple project to test couchDB functionalities. The main objective was to test offline capabilities and sync when it back on-line!

And yeah, it works! Perfectly :D

## Stack
- React
- Nextjs
- CouchDB
- PouchDB
- TypeScript
- CSS Modules
